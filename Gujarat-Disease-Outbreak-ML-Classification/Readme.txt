Gujarat Disease Outbreak Classification 2024
Overview

This project focuses on classifying disease outbreaks in Gujarat into:

Food-borne
Vector-borne
Waterborne

using machine learning and multi-source epidemiological data.

Methodology (High-Level)

Feature Selection (Mutual Information)
Feature Engineering (WHO-based indicators)
Seasonal Encoding (Fourier Transform)
Class Imbalance Handling (SMOTE)
Model: Gradient Boosting

Results
Accuracy: 79%
Macro F1 Score: 0.718
Significant improvement in minority class detection

Key Insights
Water quality plays a critical role
Historical disease patterns dominate predictions
Feature engineering significantly improves performance
